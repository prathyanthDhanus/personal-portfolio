Fullstack Project

Laundry App

This is a user-friendly web application designed to streamline the laundry process.Users can easily schedule laundry pickups and deliveries from the convenience of their laptop or smartphones.
Designed a modular-based architecture with Node.js, Express.js, and MongoDB, improving system performance metrics by 85% through increased code reusability and readability.
In the frontend, I used React.js with Tailwind CSS to create a clean UI. I provided a private route for route protection to block unauthorized entries. For the backend, I used Nodemailer for sending OTPs and JWT for authentication.

  • React.js
  • Tailwind CSS
  • Node.js,Express.js
  • MongoDB
  • Nodemailer
  • Json Web Tokens

Cake Story

It is a website for a bakery, specializing in cakes. Users can easily order their favorite cakes using this website. This website is built with React.js and React Bootstrap.
